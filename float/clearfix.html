Clearfix Demonstration

This page demonstrates using the so-called "clearfix hack": 

.clearfix::after {
  content: "";
  clear: both;
  display: table;
}

This ruleset is applied to a parent block element when all child block elements are floating. Normally, in this situation, the parent item would have no height, and so it collapses vertically.

You can also apply this property to a ruleset, if you're too lazy to remember the example above (I have to look it up every time): 

    overflow: auto;
