Cookbook >> CSS Positioning

Here is a list of recipes that demonstrate various types of positioning.

Basic fixed menu
A basic fixed menu using CSS positioning
Sticky menu and footer
Demonstrates a page with a hero image with sticky menus and a sticky footer.
Sticky headings
Demonstrates applying the position: sticky property to multiple headings in a document. The headings stick to the top of the page and overlap one and other as you scroll down.
Z-Index
A demonstration of using CSS's z-index property to override the default ordering that the browser would use. Normally, you should not need to set this property. Instead, you should order your HTML elements so that the browser performs this task for you. However, when writing components in frameworks like React, Angular, or Vue, you don't always have control over how your HTML gets ordered.
